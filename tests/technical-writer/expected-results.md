# Expected Results

## readme

Expected characteristics:
- overview appears near the top
- quick start appears before deep detail
- usage examples exist
- troubleshooting exists

## api-doc

Expected characteristics:
- parameter and return sections are explicit
- at least one example exists
- error cases are documented

## incomplete-source

Expected characteristics:
- assumptions are labeled
- unsupported specifics are avoided
- the document remains actionable where possible
